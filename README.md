# Fretboard Lab

An offline chord voicing lab for finding playable guitar shapes across tunings.

Product page: https://hannes-software.com/fretboard-lab/
Downloads: https://hannes-software.com/fretboard-lab/#downloads
Latest release: https://github.com/hannes-wan/fretboard-lab-official/releases/tag/v0.1.3
